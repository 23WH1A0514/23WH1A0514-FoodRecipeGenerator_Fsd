# 🍽️ Food Recipe Generator – Full Stack Application

## 📖 Overview
The Food Recipe Generator is a full-stack web application that helps users generate and explore food recipes based on ingredients or user input. The application provides recipe details such as ingredients and cooking instructions through an interactive and user-friendly interface.

This project is built using a client–server architecture, where the frontend handles user interaction and the backend manages data processing and API requests.

## 🚀 Installation and Setup

### Prerequisites
- Node.js
- npm
- Git

---

### Backend Setup
cd backend  
npm install  
node server.js  

Backend runs on: http://localhost:5000

---

### Frontend Setup
cd frontend  
npm install  
npm start  

Frontend runs on: http://localhost:3000
